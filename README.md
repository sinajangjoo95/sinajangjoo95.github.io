# Sina Jangjoo Academic Website

This repository contains the single-file HTML website for Sina Jangjoo's academic website.

## Files

- `index.html` — the full website code. Upload this to the root of your GitHub repository.
- `.nojekyll` — tells GitHub Pages not to process the site with Jekyll.

## Deploy with GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `.nojekyll` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Click **Save**.

Your site will publish at:

`https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Custom domain

If you want to use a custom domain, add the domain in **Settings → Pages → Custom domain** after DNS is configured.
