# Option A: Upload through GitHub website
# 1. Create a new repository on GitHub.
# 2. Click Add file → Upload files.
# 3. Upload index.html and .nojekyll.
# 4. Commit changes.
# 5. Enable GitHub Pages under Settings → Pages.

# Option B: Upload with Git command line
# Replace YOUR-USERNAME and YOUR-REPOSITORY with your GitHub details.

git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git
cd YOUR-REPOSITORY
cp /path/to/index.html ./index.html
touch .nojekyll
git add index.html .nojekyll
git commit -m "Add academic website"
git push origin main
